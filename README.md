# Project Title

How do demographic factors (age, sex, race) in missing persons data differ between Pennsylvania and Ohio?

## Overview

This project addresses the research question above using the NamUS dataset, specifically exploring how age range, biological sex, and race/ethnicity demographics differ in the missing persons between the two states. The goal of this project is to compare the two states (Pennsylvania and Ohio) to conclude which state has more missing people of a certain demographic.
